I created this website to serve as a way to access my home built NAS from anywhere in the world. In order to do that I figured that building this website was the best way to go about it! I built this website form the ground up, starting with the html and CSS files, moving into the banner, logo, and other picture designs before finally figuring out how to even host a website, and then making sure it had all the bells and whistles. Additionally I bought this domain and host the website from the NAS itself.

This website now serves many more functions than what it orginally was intended for as it has grown and morphed since its orginal conception and now serves as a personal protfolio and a solution to a problem that me and my friends have had for a while, which is how to swap files and folders amongst each other. This program is free to use, colelcts no data, and requires no account. Just upload and share!
